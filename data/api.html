What is an API Key?

An API key is a unique identifier that allows developers to access the Librebook API securely. It acts as a password for programmatic requests.

Generating an API Key

To generate a new API key:

  1. Login to your Librebook account.
  2. Navigate to Settings → API key hub.
  3. Click Create New API Key.
  4. This key is unique to your user and can be viewed again on this page.

Using the API Key

The Librebook API system has multiple different methods that includes the following:

  • Send a message from your account 
    send_message
  • Get a list of all the public messages you have sent 
    getall_message
  • Get a list of the people you follow 
    people_you_follow
  • Send a direct message to another user 
    dm_sendmessage

API request format

To use your api key, simply add this onto the end of the url of the instance you made your key on (e.g. librebok.co.uk).

[url]/api.php?apikey=[API KEY HERE]&function=[FUNCTION ID HERE]

Example:

[url]/api.php?apikey=[API KEY HERE]&function=getall_message

Security Best Practices

  • Never share your API key publicly.
  • Do not store keys directly in frontend code.

Revoking an API Key

If your key is compromised or no longer needed, revoke it:

  1. Login to your Librebook account.
  2. Navigate to Settings → API key hub.
  3. Click Create New API Key.
  4. This will generate a new key that will replace your old one.
