Control Your Precence

Librebook's main feature is the freedom it gives to it's users. One of these freedoms is the ability it gives it's users to be forgotten. This page will explain how to delete all messages that you have sent on Librebook.

Deleteing All Messages

To delete all messages:

  1. Login to your Librebook account.
  2. Navigate to Settings â†’ User Data Center.
  3. At the top of the page will be a large red button labelled "Delete All Messages".
  4. Press this button to delete all messages that you have sent. Warning! This action is not reversible!

Developer information

The all messages deletion system works by printing an input in a form under the message at the top of the page. In the Librebook SQL structure, the messages are stored with the format ID | Name | Message | TIMESTAMP | NSFW. The PHP scripting deletes all messages sent from the user. Below is the relevant semgent of PHP script:

<$sql = "DELETE FROM messages WHERE name = :user_name"; \n $stmt = $pdo->prepare($sql); \n $stmt->bindParam(':user_name', $user_name, PDO::PARAM_STR);'>;
